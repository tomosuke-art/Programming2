# Programming2
## C言語とは  
・オブジェクト指向なし<br>
・多数のおまじないが必要<br>
・最近のものと比べるとライブラリは充実していない<br>
・「;」は必ず必要<br>

## コンパイルのやり方  
```
gcc -o ファイル名 ファイル名.c
```
**実行**<br>
```
./ファイル名
```

## Scanf関数  
printf関数と違いデータ型を厳格に定義<br>
キーボードを使い値を入力できる<br>

## アルゴリズム  
### 選択ソート  
1番目の値から最後の値までの中の最小値を見つけ出し、1番目の要素と交換する。  
次に、2番目から最後の値までの中の最小値を見つけ出し、2番目の要素と交換する。  
次に3番目の…」という手順を繰り返してソートしていくアルゴリズム
